# Nodejs, Typescript starter

## Built with

- [x] Typescript
- [X] MongoDB
- [x] Express

## Features

- [x] Authentication
- [x] Email verification
- [X] Cors
- [X] Folder structure
- [X] Documentation (coming soon)
- [X] Testing (coming soon)
- [X] Docker


## Clone the repo

```bash
git clone https://github.com/Mutesa-Cedric/node-ts-starter.git
```

## Install

with npm :

```bash

npm install

```

with yarn :

```bash

yarn install

```

## Run

```bash
yarn dev
```

or

```bash
 npm run dev
```

## Build

```bash
yarn build
```

or
```bash
npm run build
```
